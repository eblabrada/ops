# OPS Technical Documentation

## C++ code style

For C++ codes, our approach is to follow [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html) wherever possible.

### Indentation

Use only spaces, and indent 4 spaces at time. You should set your editor to emit spaces when you hit the tab key.

## Python code style

For Python codes, our approach is to follow [PEP 8](https://peps.python.org/pep-0008/) wherever possible.
